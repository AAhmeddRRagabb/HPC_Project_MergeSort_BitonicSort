>> to compile the function
g++ -fopenmp main.cpp functions/merge_sort.cpp functions/utils.cpp -o app.exe

>> to run the app
./app.exe --impl serial --threads 1 --size 1000000 --repeats 5 --output results.csv
[impl -> specify the implementation type serial | openmp]
[threads -> specify the number of threads]
[size -> specify the array size]
[repeats -> for calculating average]
[output -> specify the file to store the output, to record all results keep it results.csv]
